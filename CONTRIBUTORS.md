## How to Contribute

We welcome contributions from everyone. Here is how you can help:

1. Add new applications by submitting YAML configurations
2. Report issues or suggest improvements
3. Improve documentation
4. Help with code reviews

## Recognition

All contributors who have submitted pull requests that were merged will be listed here.

## Adding Your Application

To add your application to Buge Store:

1. Create a new directory under `apps/` with your package name
2. Copy the template from `templates/app-template.yml`
3. Fill in your application details
4. Submit a Pull Request

Our automated system will validate your configuration and merge it if everything checks out.
